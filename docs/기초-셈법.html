Chapter 1 기초 셈법

1.1 사칙연산

tibble(
  기호 = c("[+]", "[-]", "[*]", "[%]"),
  설명 = c("덧셈", "뺄셈", "곱셈", "나눗셈")
) %>% kable()
기호 설명
[+] 덧셈
[-] 뺄셈
[*] 곱셈
[%] 나눗셈

1.2 기초 기술통계

tibble(
  명령어 = c("sum", "mean", "max", "min"),
  설명 = c("총합", "평균", "최대값", "최소값")
) %>% kable()
명령어 설명
sum 총합
mean 평균
max 최대값
min 최소값
